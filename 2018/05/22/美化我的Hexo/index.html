美化我的Hexo

Veröffentlicht am 2018-05-22 
美化我的Hexo

参考教程：
https://blog.csdn.net/Hoshea_chx/article/details/78826689

1 . 在站点根目录输入  （下载主题）

#原博客的主题版本是旧的，新的这样下载
$ cd hexo
$git clone https://github.com/theme-next/hexo-theme-next themes/next

#会发现在hexo/theme目录下多了个next的文件夹（我站点在hexo文件夹内）

2 . 完成后，打开 站点配置文件， 找到 theme 字段，并将其值更改为 next

效果： theme：next

#注意，：后面要留一个空格

3 . 在终端输入

hexo clean

#清除缓存 ，这样会把/hexo/public 文件夹删除

hexo g

#重新生成代码

hexo s -p 5000

#部署到本地，指定端口，因为我的4000端口用不了，所以我就改了下。执行完之后就可以在浏览器输入地址 localhost:5000 看到效果图了

hexo d

#推到github

然后就可以在 yourname.github.io 看到效果了

4 . 配置next

nexT主题有三种选择：
Muse - 默认 Scheme，这是 NexT 最初的版本，黑白主调，大量留白
Mist - Muse 的紧凑版本，整洁有序的单栏外观
Pisces - 双栏 Scheme，小家碧玉似的清新

在 站点根目录/themes/next/_congig.yml 文件中修改，搜索关键词schemes：

scheme: Pisces

改成这样的效果，然后执行hexo clean ,hexo g,hexo s -p 5000进行预览，在浏览器输入地址 localhost:5000进行预览，没问题的话就可以执行 hexo d 推到github了。

5 . 拓展博客页面设置

接下来的教程参考：9%E6%80%81%E5%8D%9A%E5%AE%A2%E6%90%AD%E5%BB%BA+%E4%B8%AA%E4%BA%BA%E5%AE%9A%E5%88%B6/

编辑配置文件，在站点根目录文件夹/_config.yml

#建议使用Notepad++ 或 Sublime Text 之类的文本编辑器进行编辑。

添加以下内容：

Extensions # 以下为第三方插件设置

#插件安装：
命令
$ npm install –save hexo-generator-search
$ npm install –save hexo-blogroll

Plugins: https://hexo.io/plugins/

Themes: https://hexo.io/themes/

theme: next # 使用的主题，主题配置后面讲
search: # 本地搜索插件
path: search.xml
field: post
format: html
limit: 10000
feed: # RSS订阅插件
type: atom
path: atom.xml
limit: 0

接着设置next主题配置

修改了next的主题设置不用执行hexo clean，直接hexo g 就可以生效了
官方链接;http://theme-next.iissnan.com/faqs.html

如何设置「阅读全文」？

在首页显示一篇文章的部分内容，并提供一个链接跳转到全文页面是一个常见的需求。 NexT 提供三种方式来控制文章在首页的显示方式。 也就是说，在首页显示文章的摘录并显示 阅读全文 按钮，可以通过以下方法：

在文章中使用 手动进行截断，Hexo 提供的方式 推荐
在文章的 front-matter 中添加 description，并提供文章摘录
自动形成摘要，在 主题配置文件 中添加：

auto_excerpt:
enable: true
length: 150
默认截取的长度为 150 字符，可以根据需要自行设定

建议使用 （即第一种方式），除了可以精确控制需要显示的摘录内容以外， 这种方式也可以让 Hexo 中的插件更好的识别。
